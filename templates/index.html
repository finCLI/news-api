# What is finCLI

As it says bringing Finance Updates on your Command Line Interface. Hence the name finCLI!
Get all the updates on BTC, Ethereum, DogeCoin, SENSEX, NIFTY, General Finance News - India & World via your CLI.

# Why did we build our own NEWS API

Initially we planned to use Currents API which does have a limited API requests per month, which lead to thinking about since we wanted to deliver the latest update on the following topics to the user. Another challenge was the information has to be credible enough therefore verified and trusted sources only.

More on API Usage? Explore the docs

# Endpoints

GET

  1. /bitcoin : updates on Bitcoin
  2. /ethereum : updates on Ethereum
  3. /cryptocurrency : updates on Cryptocurrency
  4. /globe : updates on Global Finance
  5. /india : updates on Indian Finance
